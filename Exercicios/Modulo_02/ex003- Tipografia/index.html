Tipografias

Estudo de qual forma de letras usar.

Cada letra de uma fonte é chamada de glifo

O conjunto de glifos é a fonte.

Algumas fontes tem famílias tipográficas com variações de uma mesma letra.

Alturas:

  • Altura X: Altura da letra minúscula, baseada na altura do x minúsculo.
  • Altura das letras maiúsculas
  • Altura Descendente: Das letras que vazam para baixo como o "g" ou o "q"
  • Altura Ascendente: Altura das letras que vazam pra cima como "b" ou "d"
  • Corpo: É a altura total

Itens anatômicos geométricos:

  • Serifa: Pezinho da letra. gera um alinhamento das palavras e torna o texto mais alinhado.
  • Haste: região entre o pé e cabeça da letraa.
  • Filete: Uni duas hastes linearmente
  • arco: uni curvilinearmente duas hastes
  • esporão:
  • Vértice
  • Terminal
  • Braço, sai de uma haste ou de outro braço
  • perna
  • pé
  • espinha
  • barriga
  • Olho: buraco dentri=o das letras.
  • Orelha
  • Cauda

Categorias de Fontes! (font-family)

  • Serifadas: Para textos longos impressos.
  • Sans-serif: Para texto longos virtualmente
  • Monoespaçada: Todas as letra ocupam o mesmo espaço lateral(largura), incluindo virgula e espaços
  • Handwriting: Simulação da Grafia Humana
  • Display: Personalizado

Safe Combinations:

Conjunto de fontes parecidas que funcionam como titular e reservas. Se a primiera escolha nn for possível de representar, é tentado a segunda, terceira, quarta...

É importante terminar as safes Combinations com uma das 5 categorias de fontes vistas anteriormente, pois assim o próprio computador consegue assimiliar um tipo genérico de fonte.

Existe uma lista no site: w3schools

Tamanhos - Size

A recomendaçõa oficial é o uso do px e em. Relação: 16px = 1em (na maiorida dos casos).

O tamanho padrão dos navegadores é 16px

  • Medidas absolutas

    • cm
    • mm
    • px
    • in
    • pt
    • pc

  • Medidas relativas

    • em - Altura do M maiúsculo
    • ex - Altura do x minúsculo
    • rem - altura do M maisúsculo relativo á fonte atual.
    • vw - Largura da tela
    • %
