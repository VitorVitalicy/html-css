Usamos o formato tag#seletor para persolanizar um tipo específico de tag

Ou podemos usar também apenas #seletor para aplicar essa personalização a qualquer tag.

Para usar um seletor único, usamos o id mas para personalizar mais do que uma linha de código é formalmente correto usarmos o class

Ainda sim é possível usar mais de uma classe em uma única tag em html. basta, dentro da aspas, separá-las por espaços. Ex: class:"destaque principal"

Hierarquia

Em sobrecarga de métodos. sebropõe-se aquele seletor que foi aplicado por ultimo.

PseudoClasses

Pseudo classes tem a ver com o estado semântico de uma tag. Exemplo: ativo, inativo, clicado, mouse em cima, carregando, etc...

Algumas Pseudoclasses e suas funções...

  • hover: Apresenta sua funcionalidade quando o mouse está sobre a marca gráfica específica.
  • Visited: Quando um link já foi clicado
  • Active: Quando uma tag está no ato de seu funcionamento. No caso da tag a por exemplo, ocorre quando ela está precionada.

PseudoElementos

Pseudo elementos são opbjetos criados por classes ou pseudo classes que agem diretamente na função dela.
