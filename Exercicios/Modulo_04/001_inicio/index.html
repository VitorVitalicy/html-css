O que é o Iframe?

Conteúdo inline que permite visibilidade de mídia local ou remota.

Este é um exemplo de Iframe

Detalhes importantes !

  • Cuidado com os direitos autorais po!
  • Existem alguns sites com proteção para não serem acessados via iframe como por exemplo o próprio site do google e o youtube.
  • Dentro da tag iframe é possível colocar informações que aparecerão somente quando não for possível abrir o iframe por motivos de compatibilidade.
  • Tamanho padrão de 300 x 150 (largura x altura)
  • O parâmetro scolling é sobre a presença da barra de rolagem e admite valores: auto, yes, no
  • O metodo frameborder é binário e aceita zero ou não zero.

Iframe com navegação interna!

Temos 3 link que abrirão no iframe abaixo:

  • Primeiro link
  • Segundo link
  • Terceiro link

Para isso, nomeamos um iframe com ( name="...") e sincronizamos com a tag de link através do metodo ( target="name")

Podemos redirecionar links para o iframe e ainda ter um link inicial padrão nele -- usando o metodo srcdoc podemos inseir conteúdo html dentro do espaço iframe.
