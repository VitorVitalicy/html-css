Início do CSS

Existem três formas de Aplicar as CSS

  1. Inline >> Apenas para alterações específicas e locais. Cuidado pois pode poluir o código
  2. Estilos Interno (Em bloco) >> Para Modificações específicas página por página. Fica na parte da cabeça do html.
  3. Estilos Externos >> Aplica-se a várias páginas ao mesmo tempo. Trata-se de criar um arquivo .css que contem todos os estilos e ancorar na cabeça de cada página html. Ainda assim é possível ancorar mais de um link de .css
