Links Externos

  • Puxamos a tag de ancoragem "a" com seu método principal "href" que descreve o link de acesso.
  • O primeiro abre um link que sobrepõe o nosso site.
  • Usando os métodos "target = _blank" e "rel = external", o segundo link irá abrir em outra guia.
  • A segunda forma é a mais recomendada.
  • Para links sem o aval do desenvolvedor ou propagandas patrocinadas, recomenda-se usar o método "rel = nofollow" que restringe opções dinâmicas de navegação. Por isso que na hora de anunciar um produto existe a opção nofollow e defollow.

Links Internos

  • Ligação entre páginas do próprio servidor
  • Normalmente páginas do mesmo site se sobescrevem e não abrem uma nova guia. Para isso usaremos o comando "target = _self"
  • O primeiro exemplo é de uma página que pertence ao mesmo diretório da principal
  • O segundo exemplo é de uma página que está em outro diretório do explorador de arquivos.
  • Para navegar entre paginas internas, usado ainda o método:
    • "rel=next" para mencionar ao Google que a página linkada é a próxima na hierarquia do seu site
    • "rel = prev" para mencionar ao google que a página linkada é a imediatemente anterior na hierarquia do seu site

Links para Downloads

Seguindo a mesma lógica:

  • Escreveremos em link todo o endereço do arquivo a ser baixado
  • na tag "a" usaremos o o método "download" seguido do nome do arquivo
  • Ainda usaremos depois o método "type" com a assinatura ideal para cada tipo de arquivo (pdf, zip, mp3...)
  • A lista com todas as assinaturas está nesse site: https://www.iana.org/assignments/media-types/media-types.xhtml
