Imagens Dinâmicas

Aprenderemos como adionar imagens que se adaptam com o tamanho do dispositivo (celular, tablet, notebook, TV)

Alguns detalhes...

  • Usaremos a tag picture e detro dela a tag source
  • Deve haver uma ordem correta entre as várias teclas source que serão geradas. Nesse caso, que usamos o max-width, devemos aplicar da imagem menor para a maior.
  • O método media recebe o tamanho máximo ou mínimo escolhido
    • É bom que o valor mencionado seja uns 50 pixels maior (se for max) ou menor (se for min)
    • Nesse caso a tag foi a seguinte: media=(mas-width:1050)
  • A tag srcset aplica a imagem de acordo com o tamanho
    • Normalmente pensamos em 3 tamanhos diferentes
  • A tag type aplica o tipo de mídia e o formato do arquivo
    • Nesse caso o tipo de mídia foi imagem e o formato em png, logo type=image/png
